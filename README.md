## My Rotten Tomatoes

### Run the project

``` bash
# Move to project directory
cd premoview

# Install dependencies
npm install

# Set .env 
mv .env.example .env.local  #Then modify the MONGODB_URI by providing your database URI

# Launch the server
npm run dev
```
